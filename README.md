# Stubs
A SwiftUI app for storing digital representations of concert ticket stubs. This is a demonstration app that incorporate features announced at WWDC23.

<b>Stubs</b> benefits from:
* SwiftData for persistence.
* Animated SF Symbols
* New ScrollView APIs
